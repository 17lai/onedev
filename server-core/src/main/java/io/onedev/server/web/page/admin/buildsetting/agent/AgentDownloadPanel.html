Download & Install Agent
  1. Make sure Server URL in system setting is specified correctly
  2. Make sure JDK 1.8 or higher is installed
  3. Download agent package for Windows or Linux/Mac. A new agent token will be included in the package
  4. Extract the package into a folder
  5. From extracted folder, run bin/agent.sh console (Linux/Mac) or bin\agent.bat console (Windows)
  6. Check agent installation guide for more details, such as running as docker container, running as service etc.
